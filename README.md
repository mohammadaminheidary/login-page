# Login Form

A modern, interactive login form with smooth animations and pastel white theme.

## Features

- Responsive centered card layout with flexbox
- Pastel white color scheme (#F8F8FF) with pink accent (#d52d73)
- Smooth hover and focus transitions on all inputs
- Scale animations on interactive elements
- Custom placeholder styling
- Box shadow effects for depth

## Technologies Used

- HTML5
- CSS3 (Transitions, Flexbox, Hover effects)
